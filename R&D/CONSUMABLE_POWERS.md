# Consumable In-Game Powers

## Power Types

### Rotation Power
- Turn shapes for X number of turns
- Temporary ability to rotate pieces
- Duration-based or turn-count based

### Shape Swap Power
- Interchangeable selected shape for X number of turns
- Swap between available shapes
- Tactical flexibility

### Reroll Power
- Reroll queued shapes (one-time use)
- Get a new set of shape options
- Limited use makes it strategic

### Queue Expansion
- Larger shapes queue (whole game)
- Permanent upgrade for the game instance
- See more shapes ahead for better planning

## Implementation Notes
- Powers should be purchasable with currency
- Clear visual indicators when powers are active
- Consider cooldowns or limited uses
- UI for power activation and status
- Integration with existing shape selection system
